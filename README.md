# Trabalho-Pratico_Rouba-Monte
Rouba Montes (Adaptado): Implementação do jogo de cartas Rouba Montes para a disciplina de Algoritmos e Estruturas de Dados (PUC-Minas). O projeto foca no uso de estruturas de dados para gerenciar o baralho, montes, área de descarte e o histórico dos jogadores (ranking, etc.). Inclui log de ações e exibição de ranking.
